## How It Works

<Steps>
  <Step title="User Interaction">
    Users interact with agents through various channels (email, chat, WhatsApp)
  </Step>
  <Step title="Agent Processing">
    Agents process requests using:
    - Their knowledge base
    - Configured playbooks
    - Available tools and integrations
  </Step>
  <Step title="Response Generation">
    Agents generate appropriate responses based on:
    - Context understanding
    - Playbook guidelines
    - Available data
  </Step>
  <Step title="Action Execution">
    Agents perform actions like:
    - Sending responses
    - Updating systems
    - Triggering workflows
  </Step>
</Steps>
