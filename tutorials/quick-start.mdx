## Step 1: Create an Agent

<Frame>
  <img src="/images/create-agent.png" alt="Agent Creation Interface" className="rounded-lg" />
</Frame>

<Steps>
  <Step title="Navigate to Agent Dashboard">
    From the main dashboard, click the **Create Agent** button in the top right corner.
  </Step>
  
  <Step title="Configure Basic Settings">
    <CardGroup cols={2}>
      <Card title="Identity">
        <ul className="list-disc list-inside space-y-1 mt-2">
          <li>Name your agent (e.g., "Support Assistant")</li>
          <li>Add a description of its purpose</li>
          <li>Select agent type (support, sales, etc.)</li>
        </ul>
      </Card>
      <Card title="Communication">
        <ul className="list-disc list-inside space-y-1 mt-2">
          <li>Set up email address</li>
          <li>Add phone number (optional)</li>
          <li>Configure messaging channels</li>
        </ul>
      </Card>
    </CardGroup>
  </Step>
  
  <Step title="Set Up AI Tools">
    <CardGroup cols={2}>
      <Card title="AI Models">
        <ul className="list-disc list-inside space-y-1 mt-2">
          <li>Select AI providers (ChatGPT, Claude)</li>
          <li>Configure response parameters</li>
          <li>Set up system instructions</li>
        </ul>
      </Card>
      <Card title="Knowledge Base">
        <ul className="list-disc list-inside space-y-1 mt-2">
          <li>Upload reference documents</li>
          <li>Add domain-specific knowledge</li>
          <li>Set response guidelines</li>
        </ul>
      </Card>
    </CardGroup>
  </Step>
</Steps>
