## Understanding the Code Editor

The Code Editor provides access to the files that define your agent's behavior, appearance, and functionality:

<AccordionGroup>
  <Accordion title="File Structure" icon="folder-tree" defaultOpen>
    - **src/**: The main source directory
      - **components/**: UI components for your agent
      - **pages/**: Main interface files
      - **styles/**: CSS and styling files
    - **config/**: Configuration files for agent behavior
  </Accordion>
  
  <Accordion title="Key Files" icon="file-code">
    - **index.tsx**: The main entry point of your agent
    - **components/*.tsx**: Individual UI components
    - **styles/globals.css**: Global styling rules
    - **config/agent.json**: Core agent configuration
  </Accordion>
</AccordionGroup>

<Warning>
  While the Code Editor gives you powerful customization capabilities, incorrect changes could affect your agent's functionality. Consider making a backup or duplicate of your agent before significant changes.
</Warning>

## Basic Editing Operations

<Steps>
  <Step title="Access the Code Editor">
    From your agent's dashboard, click "Code" or "Code Editor" to open the editor interface.
  </Step>
  
  <Step title="Navigate Files">
    Use the file tree on the left side to browse and select files for editing.
  </Step>
  
  <Step title="Edit Code">
    Make changes directly in the editor. The editor supports syntax highlighting and basic code completion.
  </Step>
  
  <Step title="Save Changes">
    Click "Save" or press Ctrl+S (Cmd+S on Mac) to save your changes.
  </Step>
  
  <Step title="Test Your Agent">
    Use the "Test Agent" button to see how your changes affect your agent's behavior.
  </Step>
</Steps>
