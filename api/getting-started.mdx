## Introduction

The Demiurg API provides programmatic access to:
- Create and manage agents
- Build playbooks
- Design workflow pages
- Monitor performance

## Authentication

1. Get API credentials in **Settings > API**
2. Keep keys secure
3. Include in requests:

```bash
curl https://api.demiurg.io/v1/agents \
  -H "Authorization: Bearer YOUR_API_KEY"
```

## Core Concepts

### RESTful Endpoints

- Standard HTTP methods
- Predictable URLs

```
GET /agents
POST /playbooks
PUT /pages/{id}
```

### JSON Responses

Consistent JSON structure:

```json
{
  "id": "ag_123456",
  "name": "Support Bot",
  "created_at": "2025-01-15T..."
}
```

## Code Examples

### Node.js

```javascript
const axios = require('axios');

const demiurgClient = axios.create({
  baseURL: 'https://api.demiurg.io/v1',
  headers: {
    'Authorization': `Bearer ${DEMIURG_API_KEY}`,
    'Content-Type': 'application/json'
  }
});

async function createAgent() {
  try {
    const response = await demiurgClient.post('/agents', {
      name: 'Sales Assistant',
      type: 'sales',
      communication: {
        email: 'sales@company.com',
        channels: ['email', 'chat']
      }
    });
    
    return response.data.id;
  } catch (error) {
    console.error('Error creating agent:', error.response?.data);
  }
}
```

### Python

```python
import requests
import json

BASE_URL = 'https://api.demiurg.io/v1'
headers = {
    'Authorization': f'Bearer {DEMIURG_API_KEY}',
    'Content-Type': 'application/json'
}

def create_agent():
    payload = {
        'name': 'Sales Assistant',
        'type': 'sales',
        'communication': {
            'email': 'sales@company.com',
            'channels': ['email', 'chat']
        }
    }
    
    response = requests.post(
        f'{BASE_URL}/agents',
        headers=headers,
        data=json.dumps(payload)
    )
    
    if response.status_code == 200:
        return response.json()['id']
    else:
        print(f"Error: {response.text}")
        return None
```
