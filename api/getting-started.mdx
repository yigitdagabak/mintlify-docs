## Quick Start

The Demiurg API enables programmatic access to:
- Create and manage agents
- Build and run playbooks
- Design workflow pages
- Monitor system performance

## Authentication

1. Generate API credentials in your Demiurg dashboard
2. Securely store your keys
3. Include the API key in request headers

```bash
curl https://api.demiurg.io/v1/agents \
  -H "Authorization: Bearer YOUR_API_KEY"
```

## Core Concepts

### Request Structure

- RESTful endpoints
- JSON request and response bodies
- Standard HTTP methods (GET, POST, PUT, DELETE)

### Response Format

```json
{
  "id": "ag_123456",
  "name": "Support Agent",
  "created_at": "2025-01-15T12:00:00Z"
}
```

## Code Examples

### Create an Agent (Node.js)

```javascript
const axios = require('axios');

const client = axios.create({
  baseURL: 'https://api.demiurg.io/v1',
  headers: {
    'Authorization': `Bearer ${API_KEY}`,
    'Content-Type': 'application/json'
  }
});

async function createAgent() {
  try {
    const response = await client.post('/agents', {
      name: 'Customer Support Bot',
      type: 'support',
      communication: {
        channels: ['email', 'chat']
      }
    });
    return response.data;
  } catch (error) {
    console.error('Agent creation failed', error);
  }
}
```

### Create an Agent (Python)

```python
import requests

BASE_URL = 'https://api.demiurg.io/v1'
headers = {
    'Authorization': f'Bearer {API_KEY}',
    'Content-Type': 'application/json'
}

def create_agent():
    payload = {
        'name': 'Sales Assistant',
        'type': 'sales',
        'communication': {
            'channels': ['email', 'chat']
        }
    }
    
    response = requests.post(
        f'{BASE_URL}/agents', 
        headers=headers, 
        json=payload
    )
    
    return response.json()
```
