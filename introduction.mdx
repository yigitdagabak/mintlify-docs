## Platform Overview

Demiurg consists of two main interfaces: the Agent Dashboard for managing your automated agents, and the Playbook Editor with integrated AI assistance.

<CardGroup cols={2}>
  <Card title="Agent Dashboard" icon="gauge">
    A central hub where you:
    - Create and manage agents
    - Assign playbooks to agents
    - Monitor active conversations
    - Track performance metrics
  </Card>
  <Card title="Playbook Editor" icon="pen-to-square">
    TipTap-based editor where you:
    - Write playbooks in natural language
    - Get real-time AI assistance
    - Use specialized blocks (Listen, Generate)
    - Preview and test playbooks
  </Card>
</CardGroup>
