## Key Relationships

<div className="bg-gray-50 dark:bg-gray-800 p-4 rounded-lg border-l-4 border-[#598a9c] mb-6">
 <h4 className="font-medium mb-2 text-[#1C3943] dark:text-[#598a9c]">Component Relationships</h4>
 <ul className="list-disc list-inside space-y-1">
   <li>Agents are assigned one or more Playbooks</li>
   <li>Playbooks contain multiple Pages</li>
   <li>Pages are built with Listen and Generate blocks</li>
   <li>Each agent can have unique knowledge, tools, and configurations</li>
   <li>Pages can be shared across multiple Playbooks</li>
 </ul>
