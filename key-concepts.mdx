## Key Relationships

<div className="p-4 rounded-md bg-white dark:bg-transparent border-l-2 border-[#1C3943] mb-6">
  <h4 className="font-medium mb-2 text-[#1C3943] dark:text-white">Component Relationships</h4>
  <ul className="space-y-1 pl-5 list-disc text-slate-600 dark:text-slate-300">
    <li>Agents are assigned one or more Playbooks</li>
    <li>Playbooks contain multiple Pages</li>
    <li>Pages are built with Listen and Generate blocks</li>
    <li>Each agent can have unique knowledge, tools, and configurations</li>
    <li>Pages can be shared across multiple Playbooks</li>
  </ul>
</div>
