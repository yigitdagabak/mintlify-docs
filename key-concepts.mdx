## Agents

<CardGroup cols={2}>

## Playbooks

<CardGroup cols={2}>
 <Card title="Structure" icon="book-open">
   Playbooks contain:
   - Listen blocks for events
   - Generate blocks for AI processing
   - Natural language instructions
   - Error handling
 </Card>
 <Card title="Creation" icon="wand-magic-sparkles">
   Two ways to create:
   - Write manually in the editor
   - Generate with Demiurg AI
   - Or combine both approaches
 </Card>
</CardGroup>

## Block Types

<AccordionGroup>
 <Accordion title="Listen Blocks" icon="ear-listen">
   Configure what your agent should listen for:
   - Incoming messages
   - System events
   - Scheduled triggers
   - Custom events
 </Accordion>
 
 <Accordion title="Generate Blocks" icon="sparkles">
   Process events using AI:
   - Generate responses
   - Make decisions
   - Transform data
   - Execute actions
 </Accordion>
</AccordionGroup>
